 LensFlair Photography Website
A professional, responsive photography portfolio website built with pure HTML, CSS, and JavaScript.
## Features
 Modern, elegant design with smooth animations
 Fully responsive across all devices
 Interactive portfolio gallery with filtering
 Testimonial slider with auto-rotation
 Professional service showcase
 Contact form with validation
Social media integration
 SEO-friendly structure

Technologies Used
- HTML5
- CSS3 (Grid, Flexbox, Animations)
- Vanilla JavaScript
- Font Awesome Icons
- Unsplash Images

 Sections
1. Hero- Compelling introduction with CTAs
2. About- Professional background & skills
3. Services- 6 photography service offerings
4. Portfolio - Filterable image gallery
5. Testimonials - Client reviews slider
6. Contact - Contact form & information
7. Footer*- Professional closure

Usage
Open `index.html` in any modern web browser to view the complete photography portfolio website. Perfect for photographers looking to showcase their work online!

Customization
Easily customizable by:
- Replacing images with your own work
- Updating contact information
- Modifying color scheme in CSS variables
- Adding your own services and testimonials
